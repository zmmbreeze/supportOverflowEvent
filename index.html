useOverflowEvent

Way to detect overflow event support and use it with graceful degradation. Overflow event is supported on Chrome(tested on 9+) and Firefox(tested on 1.5+).

Tweet
Add some text here to make it `overflow`.

HOW TO USE

useOverflowEvent(function (addOverflowListener) {
    if (!addOverflowListener) {
        // browser not support overflow event
        // do somethings or nothing
        return;
    }

    var container = document.getElementById('demo');
    addOverflowListener(container, 'over', function () {
        console.log('overflow');
    });
    addOverflowListener(container, 'under', function () {
        console.log('underflow');
    });
});
